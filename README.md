## Open Source KMP Library that recalls the input history of specific text fields entirely on the user's device.
# Katch 

Katch is a lightweight Kotlin Multiplatform (KMP) library built for Compose Multiplatform (Android, Web/Wasm, and Desktop). It automatically tracks, caches, and recalls the input history of specific text fields entirely on the user's device.

## Features
* **Cross-Platform:** Shared logic across Android, Desktop (JVM), and Web (Wasm).
* **Local Cache:** Remembers text fields without reliance on cloud databases.
* **Privacy First:** Data never leaves the device.
